Quadratics

Quadratic Formula
x = (−b ± √(b²−4ac)) / 2a
Discriminant: >0 → 2 real; =0 → 1 real; <0 → 2 complex
Vertex Form
f(x) = a(x − h)² + k
vertex = (h, k), axis: x = h
Standard → Vertex
h = −b/2a, k = f(h)
Completing the Square
x²+bx → (x + b/2)² − (b/2)²

Complex Numbers — Powers of i

Powers cycle of 4
i¹ = i
i² = −1
i³ = −i
i⁴ = 1 (then repeats)
For iⁿ: divide n by 4, use remainder: 0→1, 1→i, 2→−1, 3→−i
Dividing Complex Numbers
Multiply by conjugate of denominator.
(a+bi)(a−bi) = a² + b²
The denominator becomes a real number

Polynomial Factoring Formulas

Difference of Squares
a² − b² = (a+b)(a−b)
Sum of Cubes
a³+b³ = (a+b)(a²−ab+b²)
SOAP: Same, Opposite, Always Positive
Difference of Cubes
a³−b³ = (a−b)(a²+ab+b²)
SOAP: Same, Opposite, Always Positive
FTA
Degree n → exactly n zeros
(counting multiplicity)
Complex zeros come in conjugate pairs
Rational Root Theorem
±(factors of constant) / (factors of leading coeff)
Remainder Theorem
f(x) ÷ (x−c) → remainder = f(c)

End Behavior (LRB)

Even, + leading
↗ ... ↗
both ends UP
Even, − leading
↘ ... ↘
both ends DOWN
Odd, + leading
↘ ... ↗
down left, up right
Odd, − leading
↗ ... ↘
up left, down right

Exponential & Log Laws

Product Rule
log(xy) = log x + log y
Quotient Rule
log(x/y) = log x − log y
Power Rule
log(xⁿ) = n · log x
Change of Base
log_b(x) = log x / log b
= ln x / ln b
Basic Properties
log_b(b) = 1
log_b(1) = 0
log_b(bˣ) = x
b^(log_b x) = x
Interest Formulas
A = P(1 + r/n)^(nt)
A = Pe^(rt) (continuous)
e ≈ 2.718

Sequences & Series

Arithmetic — explicit
aₙ = a₁ + (n−1)d
d = common difference
Arithmetic — partial sum
Sₙ = n(a₁ + aₙ) / 2
Geometric — explicit
aₙ = a₁ · rⁿ⁻¹
r = common ratio
Geometric — partial sum
Sₙ = a₁(1 − rⁿ) / (1 − r)
Infinite geometric sum
S = a₁ / (1 − r)
only when |r| < 1
If |r| ≥ 1: diverges (no sum)

Matrices

2×2 Determinant
det [[a,b],[c,d]] = ad − bc
2×2 Inverse
A⁻¹ = (1/det)·[[d,−b],[−c,a]]
Swap main diagonal, negate off-diagonal, divide by det
Solving with Matrices
AX = B → X = A⁻¹B
Order matters! Multiply A⁻¹ times B

Rational Functions — DRIVRS

D
Domain — set denominator ≠ 0, find restrictions
R
Reduce — factor and cancel common factors (cancelled factors → holes!)
I
Intercepts — x-int: set numerator = 0; y-int: plug in x = 0
V
Vertical Asymptotes — zeros of the REDUCED denominator
R
Removable Discontinuities — holes at cancelled factors; find y by plugging into reduced form
S
Sign Chart — test intervals between x-intercepts and vertical asymptotes
Horizontal Asymptotes
  • Degree top < bottom → HA: y = 0
  • Degrees equal → HA: y = (leading coeff top) / (leading coeff bottom)
  • Degree top > bottom → no HA (oblique asymptote)

Inverse Trig — Domains & Ranges

sin⁻¹(x)
Domain: [−1, 1]
Range: [−90°, 90°]
Outputs only Q1 and Q4 angles
cos⁻¹(x)
Domain: [−1, 1]
Range: [0°, 180°]
Outputs only Q1 and Q2 angles
tan⁻¹(x)
Domain: all reals
Range: (−90°, 90°)
Open interval — never reaches ±90°

Trig Graphing & Identities

General Form
y = a·sin(b(x−h)) + k
|a|=amplitude, period=360°/b, h=phase shift, k=vertical shift
Pythagorean Identity
sin²θ + cos²θ = 1
Reciprocal Identities
csc θ = 1/sin θ
sec θ = 1/cos θ
cot θ = 1/tan θ
Law of Sines / Cosines
a/sin A = b/sin B = c/sin C
c² = a²+b²−2ab·cos C
